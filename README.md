# CustomDebug[Minecraft]
Модификация для Minecraft 1.7.10-1.12
Изменяет режим отладки по кнопке F3

### Клавиши клавиатуры по умолчанию
- R-Перезагрузка ресурсов
- P-Настройки

### Отображает
- Координаты
- Биом
- Стороны света
- Температуру
- Уровень освещения
- Информацию об ОЗУ

### Режим разработчика:
- Информация о блоках
- Информация о сущностях
